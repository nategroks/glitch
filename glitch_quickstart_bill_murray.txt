1. Install it once, pray zero times
   - You need: cc, make, libpng, zlib, curl (optional), and optionally Python 3 + Pillow if you want auto color palettes.
   - From repo root:
     ./install.sh
     That:
     - Creates ~/.config/glitch/ with glitch.config, color.config, shape.config, variants/
     - Builds the binary & man page
     - Falls back to a built-in palette if Python isn‚Äôt there
   - If it‚Äôs not on your PATH:
     ./path.sh && source ~/.bashrc

2. Feed it your own pictures
   - Drop square PNGs into:
     ~/.config/glitch/variants/
   - Name them by ‚Äúnoise mode‚Äù: sunset.png, storm.png, sword.png, etc.
   - Use a specific one:
     GLITCH_VARIANT=sunset ./glitch
     (That also sets GLITCH_NOISE=sunset.)
   - Want your own image folder? Set LOCAL_IMAGES_DIR or GLITCH_VARIANT_DIR, then rerun ./install.sh.

3. Give it your favorite colors
   - Edit ~/.config/glitch/color.config (keys like BG1..BG4, FG_MEM, etc.), then:
     ./install.sh
   - Or copy a preset:
     cp color.config.nord ~/.config/glitch/color.config
     # or dracula / gruvbox / grayscale / high-contrast
     ./install.sh
   - No Python? You can still bake a palette:
     GLITCH_PALETTE=miami|sunset|neon|random ./install.sh

4. No internet, no problem
   - Kill all web PNG downloads:
     NET_IMAGES=0
     in ~/.config/glitch/glitch.config or exported.
   - For text-only:
     make minimal
     # or set NET_IMAGES=0 then ./install.sh

5. Actually run the thing
   - Normal:
     ./glitch
   - One frame and out:
     ./glitch --once
   - Lock a shape/noise:
     GLITCH_NOISE=diamond ./glitch
   - Change speed/duration:
     GLITCH_SPEED=120 GLITCH_DURATION_MS=4000 ./glitch
